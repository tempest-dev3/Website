About me

Engineering with
clarity and intent.

I'm a backend engineer who cares deeply about the craft of building software. My work lives in the layers most users never see — the APIs, the data models, the auth systems, the architecture decisions — and I think that's exactly where quality matters most.

I work primarily with Python and FastAPI, and I've developed a strong foundation in PostgreSQL, RESTful API design, and building systems that are both secure and maintainable over time. I believe good backend code is invisible — it just works, scales, and doesn't surprise you at 3am.

When I'm not writing code, I'm reading about distributed systems, exploring security concepts, and thinking about how to make software engineering a more deliberate, thoughtful practice.
