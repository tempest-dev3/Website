01 — About

Who I am

I'm Yassir Bahssou, a backend engineer building under the tempest.dev identity. My work lives server-side — designing the systems, schemas, and API contracts that products depend on.

Python is my primary language. I use FastAPI to build async HTTP APIs that are fast, properly documented, and maintainable. For JavaScript ecosystems I reach for Node.js with Express and TypeScript — type safety and a predictable codebase matter.

On the data layer I work with both worlds: PostgreSQL for relational domains — normalization, constraints, indexing, migrations — and MongoDB Atlas when document flexibility is the right call. I choose the tool that fits the data model, not the one I already know.

I'm comfortable in Linux environments: reading logs, managing processes, configuring services. My Git workflow is deliberate — feature branches, clean commits, meaningful pull requests.
